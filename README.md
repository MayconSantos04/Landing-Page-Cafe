# Landing Page Café

Landing page moderna desenvolvida para o ramo de cafeteria, projetada para expor produtos de forma atraente e facilitar os pedidos dos clientes. O projeto foi construído acompanhando o conteúdo prático da desenvolvedora Larissa Kich no YouTube, aplicando conceitos avançados de design de interface e estruturação de código.
[Acesse o link do video](https://www.youtube.com/watch?v=ik-njdH5Q5c&t=1540s)

## Funcionalidades

*   **Foco em UX/UI:** Relação visual harmoniosa voltada para o nicho de gastronomia, com navegação intuitiva para a visualização de produtos.
*   **Design Clean e Elegante:** Estilização moderna e minimalista utilizando propriedades avançadas de CSS para criar uma atmosfera aconchegante de cafeteria.
*   **Responsividade Fluida:** Interface totalmente adaptada para oferecer uma excelente experiência de navegação em smartphones, tablets e desktops.

## Tecnologias Utilizadas

*   **Front-end:** HTML5, CSS3 e JavaScript

### Execução Local

Caso queira baixar o projeto para analisar o código estruturado localmente, siga os passos abaixo:

1. Clone o repositório:
   ```bash
   git clone [https://github.com/MayconSantos04/Landing-Page-Cafe.git](https://github.com/MayconSantos04/Landing-Page-Cafe.git)

2. Navegue até o diretório do projeto:
cd Landing-Page-Cafe

3. Abra o arquivo `index.html` diretamente em seu navegador para rodar a aplicação.
